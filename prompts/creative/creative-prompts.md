# Creative Writing & Brainstorming Prompts

## Story & Fiction

### 1. Short Story Generator
```
Write a {genre} short story ({word count} words) with these elements:
- Setting: {describe time and place}
- Main character: {brief description}
- Conflict: {central problem or tension}
- Tone: {dark/lighthearted/suspenseful/melancholic/hopeful}

Requirements:
- Start in media res (in the middle of action)
- Include at least one unexpected twist
- End with a resonant final line
- Use sensory details (sight, sound, smell, touch, taste)
- Show character through action and dialogue, not exposition
```

### 2. Character Creator
```
Create a detailed character for a {genre} story:

Provide:
- **Name and age**
- **Physical description** (3-4 distinctive traits, not a police report)
- **Personality:** Core trait + contradicting trait (makes them human)
- **Backstory:** The 2-3 events that shaped who they are
- **Motivation:** What they want (external goal) and need (internal growth)
- **Flaw:** The specific weakness that creates conflict
- **Speech pattern:** How they talk (sample dialogue, 3 lines)
- **Secret:** Something the reader discovers later
- **Arc:** How they change by the end
```

### 3. Dialogue Writer
```
Write a dialogue scene between {character A} and {character B} about {topic/conflict}.

Rules:
- Each character has a distinct voice (vocabulary, sentence length, rhythm)
- Include subtext (what they mean vs what they say)
- Add minimal but evocative action beats (no "he said angrily")
- Build tension through the conversation
- End with a line that changes the dynamic

Context: {describe the situation and what each character wants}
Tone: {tense/playful/emotional/professional}
```

### 4. World Builder
```
Create a detailed setting for a {genre} story set in {era/world type}.

Include:
- **Physical environment:** Geography, climate, architecture
- **Social structure:** How society is organized, class/power dynamics
- **Technology/magic level:** What's possible and what isn't
- **Culture:** Values, customs, taboos, art, food
- **Economy:** How people make a living, trade
- **Conflict sources:** The tensions built into this world
- **Unique detail:** One surprising/original element that makes this world memorable
- **Sensory snapshot:** A paragraph describing what it feels like to walk through
```

### 5. Plot Generator
```
Generate a {genre} plot using the three-act structure:

**Act 1 — Setup (25%):**
- Opening scene that hooks
- Introduce protagonist and their normal world
- Inciting incident that disrupts everything
- Decision to act (crossing the threshold)

**Act 2 — Confrontation (50%):**
- Rising complications (3 escalating obstacles)
- Midpoint twist that raises stakes
- Allies and enemies revealed
- Darkest moment / all seems lost

**Act 3 — Resolution (25%):**
- Climax (final confrontation)
- Resolution (how the world has changed)
- Denouement (new normal)

Theme: {what the story is really about}
```

### 6. Poetry Generator
```
Write a {form} poem about {topic}.

Forms: free verse / sonnet / haiku series / limerick / villanelle / spoken word

Requirements:
- Use concrete imagery (not abstract platitudes)
- Include at least one unexpected metaphor
- Vary line length for rhythm
- Read aloud well (sound matters)
- End with the strongest line
- Tone: {contemplative/fierce/playful/haunting}
```

## Brainstorming & Ideation

### 7. Idea Generator
```
Generate 20 ideas for {topic/problem}. Constraints: {list any constraints}.

Rules:
- First 5: Safe, proven approaches
- Next 5: Creative combinations of existing ideas
- Next 5: Unconventional/contrarian approaches
- Last 5: Wild, "what if" moonshots

For each idea: one sentence description + why it might work.
Then: pick the top 3 and develop each into a paragraph.
```

### 8. Name Generator
```
Generate 30 name options for {what needs naming: product/company/feature/character}.

Context: {describe what it is, audience, tone}

Categories:
- 10 descriptive names (say what it does)
- 10 evocative names (suggest a feeling or metaphor)
- 5 invented words (portmanteau, coined terms)
- 5 one-word names (real words used metaphorically)

For each: pronunciation guide if ambiguous + available as .com (likely/unlikely).
Star the top 5 with reasoning.
```

### 9. Analogy Finder
```
Explain {complex concept} using 5 different analogies for different audiences:

1. For a 10-year-old
2. For a business executive
3. For a scientist from a different field
4. For someone who loves cooking
5. For someone who loves sports

Each analogy should: be accurate (not misleading), be memorable, highlight the most important aspect. Note where each analogy breaks down.
```

### 10. "What If" Explorer
```
Explore this "what if" scenario: {describe scenario}

Develop:
1. **Immediate consequences** (first week)
2. **Short-term effects** (first year)
3. **Long-term implications** (10 years)
4. **Who benefits and who loses**
5. **Unintended consequences** (second-order effects)
6. **The most surprising outcome**
7. **Historical parallels** (has something similar happened?)
8. **Story seed:** A character whose life this changes dramatically
```

### 11. Content Calendar Creator
```
Create a 30-day content calendar for {brand/topic} on {platform}.

For each day provide:
- Content type (educational/entertaining/promotional/engagement)
- Topic/angle
- Hook (first line)
- Key points
- CTA
- Best posting time

Mix: 40% educational, 25% engagement, 20% behind-the-scenes, 15% promotional.
Include 2 trend-jacking slots (leave flexible for timely content).
```

### 12. Presentation Outliner
```
Create an outline for a {duration}-minute presentation about {topic}.

Audience: {describe}
Goal: {what should the audience think/feel/do after}

Structure:
- **Opening hook** (30 seconds): Surprising stat, question, or story
- **Agenda** (15 seconds): What you'll cover
- **Main points** (3-4 sections): Each with a key message + supporting evidence + transition
- **Demo/Example:** One concrete demonstration
- **Objection handling:** Address the biggest skepticism
- **Closing:** Callback to opening + clear CTA

Include: slide suggestions, speaker notes, and audience interaction moments.
```

## Rewriting & Enhancement

### 13. Story Improver
```
Improve this story/passage. Specifically:
- Strengthen the opening (make it hook immediately)
- Replace telling with showing (actions and sensory details)
- Sharpen the dialogue (make each character sound different)
- Tighten the prose (cut unnecessary words)
- Improve the pacing (vary sentence length)
- Strengthen the ending (make it resonate)

Mark changes with [IMPROVED] and briefly explain why.

Text:
{paste text}
```

```
Rewrite this passage from a different perspective:

Original perspective: {first person/third person/etc.}
New perspective: {choose: first person, unreliable narrator, omniscient, second person, antagonist's POV}

Maintain the same events but let the new perspective reveal different details, biases, and emotions.

Original text:
{paste text}
```

```
Keep the same core events and character dynamics but transform:
- Setting and atmosphere
- Language and vocabulary
- Pacing and tension
- Character archetypes

Original scene:
{paste text}
```
